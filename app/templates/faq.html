FAQ

¿Que realiza la aplicación?
Es un generador de datos de prueba para sistemas ubicuos que permite trabajar con tres dimensiones de contexto: geográfico, ambiental y temporal. La aplicación utiliza modelos estocásticos para recrear situaciones reales, permitiendo a investigadores y desarrolladores evaluar el comportamiento de sistemas dependientes del entorno sin necesidad de recolección de datos en el campo.
¿Como se ingresa el código postal?
Con una de las siguientes opciones:
  1. Una letra identificatoria de la provincia, según la norma ISO 3166-2:AR.
  2. Un número de 4 dígitos que identifica la localidad, ciudad o barrio.
  3. El país de la localidad, ciudad o barrio separado por una coma (la búsqueda puede resultar más exacta si escribimos el país en inglés)
¿Que metodo de simulación se empleo?
Para simular los datos en cada contexto se empleo:
  • Contexto Geográfico: Para la generación aleatoria de puntos dentro del radio circular sobre cartografía se emplea la distribución uniforme aplicada a coordenadas polares.
  • Contexto Ambiental: Para generar los datos permite el ingreso manual de los parametros o la obtención de datos mediante la conexión con una API metereológica, en ambos casos usa la distribución uniforme para generar los datos de temperatura y la distribución normal para generar los datos de la humedad.
  • Contexto Temporal: Para generar los datos de la hora del día, se debe ingresar manualmente el rango para genera los datos usando una distribución uniforme. Para generar los datos de tiempo libre, permite el ingreso manual de los parametros para luego simular los datos usando la Transformada Inversa para Funciones Discretas o usar el Principio de Indiferencia de Laplace.
